
DATA VISUALIZATION PROJECT

Objective:
Transform raw data into meaningful visualizations to support decision-making.

Tools Used:
- Python
- Pandas
- Matplotlib

Project Structure:
- data/ : raw dataset
- scripts/ : python visualization scripts
- outputs/ : generated charts

Insights:
1. Sales show clear monthly variation.
2. Higher marketing spend generally leads to higher sales.
3. Website visitors fluctuate seasonally.

How to Run:
1. Install requirements: pandas, matplotlib
2. Run scripts inside scripts/ folder.
